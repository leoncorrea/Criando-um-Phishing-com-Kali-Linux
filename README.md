Phishing Attack
Este repositório contém um walkthrough passo a passo sobre como criar um ataque de phishing usando o Kali Linux. O conteúdo inclui desde a criação do email até a execução de uma ação social, com foco em educação e defesa contra técnicas de engano online.

Conteúdo abrangido:

Configuração inicial do Kali Linux
Criação de um e-mail de phishing
Simulação de uma ação social
Análise e mitigação de riscos
Este guia é destinado a estudantes, profissionais e entusiastas que desejam entender mais sobre segurança cibernética.

Requisitos:

Kali Linux instalado
Conhecimento básico em terminologia de redes e sistemas operacionais
Como usar o repositório:

Faça download dos arquivos ou clone o repositório no seu computador.
Siga as instruções passo a passo na pasta "walkthrough".
Execute os scripts fornecidos para realizar o walkthrough.
Nota: Este repositório é educacional e deve ser usado somente em ambientes controlados e autorizados, conforme aplicável à lei local.
